Mixinify


		@mixin constrain($width) {
			min-width: $width;
			width: 100%;
			max-width: $width;
		}

		.page-content {
			@include constrain(1000px);
		}
						

		.nav-content {
			@include constrain(1000px);
		}
eventually you move variables so far away you are in the realm of actual reuse! make some mixins! again, don't overengineer to start, keep them basic and changeable if you get 100 people involved here, they will be too generic if you only involve a few, they will be exactly as specific as they need to be
